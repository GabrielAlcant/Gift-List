#  Lista de Presentes – Casal Recém-Casado

Aplicação web desenvolvida em **React.js** para exibir uma lista de presentes para um casal recém-casado, permitindo que convidados escolham um presente de forma simples, elegante e intuitiva.

O projeto possui design moderno, layout responsivo e estrutura preparada para evoluções futuras como autenticação, pagamento e persistência de dados.

---

##  Funcionalidades

-  Exibição de lista de presentes em formato de cards  
-  Marcação de presente como escolhido  
-  Interface responsiva (desktop e mobile)  
-  Estilo moderno com Tailwind CSS  
-  Componentização com React  
-  Pronto para deploy  

---

##  Tecnologias Utilizadas

- React.js  
- Next.js  
- Tailwind CSS  
- shadcn/ui  
- JavaScript (ES6+)
